A Free Online Image Processing Website

An Image Filter is a method or procedure for altering an image's colors, shade, hue, saturation, texture, and other features. They change the appearance of photos for commercial, creative, or aesthetic purposes. We designed a web application ‘GIT Hack Image Processor’ that allows users to apply filters onto their selected image file.

Learn More about the Website

With GitHack Image Processor you can upload any image or video and apply one of the given filters. Simply upload your file, pick one of the filters, and play around with the parameters until you get the desired result. The available filters and their effects are described below.

  • Brightness: Increase or Decrease Brightness.
  • Pencil Sketch: Pencil Sketch of an Image.
  • Negative: Invert Image to create a Negative effect.
  • Color Extract: Extract pixels belonging to a specified color-range from an image.
  • Color Focus: Focus on the pixels belonging to a specified color-range.
  • Cartoon: Cartoonistic illustration of image.

The filter functionalities have been implemented using Python’s OpenCV Computer Vision library and tools. The following file formats are supported by the application: .png, .jpeg, .jpg, .bmp, .tiff'. The user is provided the option to customize the effects of their selected filters using options such as kernel size and color range.
